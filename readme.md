
# Challenge API Test Suite

This repository contains API tests with [Cypress](https://www.cypress.io/).

## ✅ Requirements

- Node.js (v18+ recommended)
- npm or yarn

## 📦 Installation

```bash
git clone https://github.com/dattademo/challenge-API.git
cd challenge-API
npm install
```


## 🤖 Headless Execution

```bash
npx cypress run
```

## ▶️ Interactive mode

```bash
npx cypress open
```
